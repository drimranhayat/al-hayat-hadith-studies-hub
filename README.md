# Al-Hayat Hadith Studies Hub

الحیات مرکزِ مطالعۂ حدیث

A static Urdu Hadith studies learning, testing, verification, research, and progress-tracking website designed for GitHub Pages.

## Key Files

- `index.html` - homepage
- `topic-index.html` - full topic index and filters
- `search.html` - client-side search
- `glossary.html` - terminology glossary
- `dashboard.html` - localStorage progress dashboard
- `assets/css/style.css` - RTL responsive design system
- `assets/js/script.js` - filters, quiz, dashboard, search, tools
- `assets/data/*.js` - structured topics, MCQs, glossary, study tools, books, templates

## Academic Rule

The site does not issue independent rulings on Hadith. Detailed scholarly content should cite reliable scholars and sources. Where verified content is not ready, use the placeholder policy in `academic-policy.html`.
