# Al-Hayat Hadith Studies Hub  
## الحیات مرکزِ مطالعۂ حدیث

**Al-Hayat Hadith Studies Hub** is a scholarly, educational, and research-oriented website dedicated to the study of Hadith, Hadith Sciences, classical Hadith collections, Hadith terminology, scholars of Hadith, sources, and research guidance.

Urdu title: **الحیات مرکزِ مطالعۂ حدیث**

Website URL:  
https://drimranhayat.github.io/al-hayat-hadith-studies-hub/

## علمی اصول

This website follows these core principles:

1. **Authenticity** — material should be based on reliable Islamic and Hadith sources.  
2. **Scholarly caution** — avoid unsourced claims, weak attribution, exaggeration, and casual internet content.  
3. **Respectful tone** — Hadith-related material must be written with dignity and reverence.  
4. **Clear Urdu** — Urdu should be natural, scholarly, smooth, and not translation-like.  
5. **Structured learning** — content should move from basic concepts to advanced topics gradually.  
6. **SEO-friendly content** — pages should be readable for users and discoverable by search engines.  
7. **Global quality** — design, language, structure, and accessibility should reflect a high-standard Islamic studies platform.

## Recommended Study Path

For new learners, the suggested order is:

1. `about.html` — understand the purpose of the platform  
2. `glossary.html` — learn basic Hadith terminology  
3. `hadith-sciences.html` — study Hadith Sciences  
4. `books.html` — understand major Hadith book categories  
5. `hadith-collections.html` — study collections and arrangements  
6. `scholars.html` — learn about Muhaddithin and their services  
7. `references.html` — understand sources and reference discipline  
8. `research.html` — move toward research guidance
