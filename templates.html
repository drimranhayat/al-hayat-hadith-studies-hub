LESSON TEMPLATE

درسِ حدیث: عملی سانچہ

  1. موضوع: ایک واضح اور محدود عنوان لکھیں۔
  2. عربی متن: صرف مستند مصدر سے نقل کیا گیا متن شامل کریں۔
  3. حوالہ: کتاب، باب، رقمِ حدیث، جلد/صفحہ اور طباعت جہاں ممکن ہو درج کریں۔
  4. مختصر ترجمہ: سلیس، فطری اور معنی کے قریب اردو ترجمہ دیں۔
  5. سیاق: حدیث کس باب یا مسئلے سے متعلق ہے؟
  6. اہم الفاظ: دو یا تین کلیدی الفاظ کی وضاحت۔
  7. علمی نکتہ: شروح یا اہلِ علم کی روشنی میں ایک بنیادی فائدہ۔
  8. عملی پیغام: اخلاقی/عملی رہنمائی، مگر متن سے آگے غیر محتاط اضافہ نہیں۔
  9. Discussion Question: ایک سوچنے والا سوال۔
  10. احتیاطی نوٹ: اگر روایت کے حکم یا فہم میں اختلاف ہو تو مختصر تنبیہ۔
TAKHRIJ TEMPLATE

تخریج الحدیث: عملی فارم

  1. حدیث کا معروف متن یا ابتدائی الفاظ لکھیں۔
  2. کلیدی عربی الفاظ منتخب کریں جن سے تلاش آسان ہو۔
  3. اصل مصادر: صحیحین، سنن، مسانید، معاجم، مصنفات وغیرہ میں تلاش کریں۔
  4. ہر مصدر کا مکمل حوالہ درج کریں: کتاب، باب، رقم، جلد/صفحہ۔
  5. طرق جمع کریں: کون سے صحابی، تابعی یا راوی کے ذریعے روایت آئی؟
  6. الفاظ کا تقابل کریں: اہم اختلافات الگ note کریں۔
  7. شواہد اور متابعات دیکھیں؛ تقویت یا ضعف کا امکان سمجھیں۔
  8. رجال کا مختصر جائزہ: کمزور یا محلِ بحث راوی note کریں۔
  9. اہلِ علم کا حکم نقل کریں، خود سے مستقل حکم نہ لگائیں۔
  10. حتمی خلاصہ: عام قاری، طالب علم یا خطیب کے لیے احتیاطی نتیجہ۔
NARRATOR PROFILE

راوی پروفائل: مختصر سانچہ

  1. مکمل نام، کنیت، نسبت اور معروف لقب۔
  2. طبقہ اور زمانہ: صحابی، تابعی، تبع تابعی یا بعد کا طبقہ۔
  3. اساتذہ: کن سے روایت کی؟
  4. تلامذہ: کن لوگوں نے اس سے روایت کی؟
  5. ائمۂ جرح و تعدیل کے اقوال۔
  6. ثقہ، صدوق، ضعیف، مدلس، مجہول وغیرہ کا خلاصہ حکم۔
  7. کسی خاص علت، تدلیس، اختلاط یا ارسال کا ذکر اگر ہو۔
  8. روایت پر اثر: اس راوی کی حالت متعلقہ حدیث کو کیسے متاثر کرتی ہے؟

ویب سائیٹ مواد بنانے کا معیاری ورک فلو

  1. موضوع منتخب کریں: عنوان محدود، واضح اور searchable ہو۔
  2. اصل مصادر جمع کریں: حدیث، شرح، مصطلح، رجال یا تخریج کے متعلقہ مصادر۔
  3. قاری کی سطح طے کریں: عام قاری، طالب علم، استاد، محقق یا تخصصی مطالعہ۔
  4. مختصر تعارف لکھیں: غیر ضروری تفصیل سے آغاز نہ کریں۔
  5. مرکزی نکات ترتیب دیں: تعریف، اہمیت، مثال، احتیاط، متعلقہ مصادر۔
  6. حوالہ جاتی نوٹ شامل کریں: روایت یا علمی دعوے کو بے حوالہ نہ چھوڑیں۔
  7. تعلیمی حصہ شامل کریں: سوال، mini quiz، discussion prompt یا revision note۔
  8. زبان review کریں: اردو فطری، سلیس، علمی اور غیر ترجمہ زدہ ہو۔
  9. links check کریں: internal links اور folder paths درست ہوں۔
  10. final upload: GitHub root یا متعلقہ folder میں exact file name کے ساتھ replace کریں۔
